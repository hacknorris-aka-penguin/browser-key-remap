paste key - one of most overused keys, mostly combination of meta+v , shift+insert, ctrl+shift+v or ctrl+v

REMAP YOUR PASTE KEY TO:

click any item from list to set ctrl + v to these keys:

  • ENTER KEY
  • BACKSPACE KEY
  • DELETE KEY
  • SPACE KEY
  • TAB KEY
  • TERMINAL NOTIFICATION
  • VERTICAL TAB KEY
  • ESC KEY
  • NOTHING
  • NON-BREAKING SPACE
  • REVERSE TEXT
  • IMPASTA

note - remapping might break some apps or output weird characters depending on OS configuration
